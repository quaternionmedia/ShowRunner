# Cookbook

The following are examples of how to use ShowRunner to manage shows, cues, and other production elements. These are meant to be used as a reference and starting point for your own projects.

## Examples

- [Self-Producing Intro Video](meta-video.md) — Complete AV recording pipeline: OBS screen capture, Ardour audio, Kokoro TTS voice-over, and Kdenlive MLT export, all managed by ShowRunner cues.

## Testing

- [UI Testing](ui-testing.md) — Page-level NiceGUI simulation tests and Playwright end-to-end tests: architecture, fixtures, writing tests for new plugins.
